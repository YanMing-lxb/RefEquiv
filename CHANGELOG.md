# CHANGELOG

## v0.1.0

### 🎉 新增功能
- 初始版本发布 - RefEquiv (制冷剂等效气相物性计算器)
- 两相制冷剂等效单相物性计算
- 支持基于 REFPROP 的物性计算
- 支持多种等效比热容方法
- 支持 McAdams 和 Cicchitti 粘度模型
- 采用 EMT 对称模型计算等效导热系数
- 支持生成 Excel 格式的物性表
- 支持命令行交互使用
- 完整的打包和发布流程
- 项目名称更新为 RefEquiv

### 📦 依赖

- numpy>=2.4.4
- pandas>=3.0.3
- openpyxl>=3.1.5
